* How to use

- clone the repo

- change directory to fcc-cli/src

- node index.js
  
#+BEGIN_SRC sh
git clone https://github.com/diegoperezm/fcc-cli.git;
cd fcc-cli/src;
node index.js 
#+END_SRC
